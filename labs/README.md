# AWS Certified Developer Associate Lab Guide

This folder contains practical labs for TGS-2025052675 - AWS Certified Developer Associate Training.

## Lab Sequence

| Lab | File | Outcome |
| --- | --- | --- |
| 1 | [Account Safety, IAM, AWS CLI, SDKs, and Developer Setup](lab-01-account-iam-cli-sdk-developer-setup.md) | Secure developer baseline with IAM, CLI, SDK, and CloudShell notes. |
| 2 | [Lambda, API Gateway, and Serverless APIs](lab-02-lambda-api-gateway-serverless-api.md) | Serverless API design with Lambda and API Gateway concepts. |
| 3 | [DynamoDB, S3, and Application Data Patterns](lab-03-dynamodb-s3-application-data.md) | Application data model and object storage access pattern notes. |
| 4 | [SQS, SNS, EventBridge, and Step Functions](lab-04-sqs-sns-eventbridge-step-functions.md) | Event-driven architecture comparison and retry/DLQ workflow design. |
| 5 | [Developer Tools and CI/CD Pipelines](lab-05-codecommit-codebuild-codedeploy-codepipeline.md) | CI/CD pipeline design using AWS developer tools. |
| 6 | [Deployment with SAM, CloudFormation, Elastic Beanstalk, and Containers](lab-06-deployment-sam-cloudformation-containers.md) | Deployment option comparison and infrastructure as code practice. |
| 7 | [Application Security, Secrets, Cognito, KMS, and Least Privilege](lab-07-security-secrets-cognito-kms.md) | Secure application configuration and identity pattern notes. |
| 8 | [Monitoring, Troubleshooting, Optimization, and Exam Review](lab-08-monitoring-troubleshooting-optimization-exam-review.md) | Observability, troubleshooting, optimization checklist, and timed exam review. |
